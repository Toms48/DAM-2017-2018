UC-0010 Pagar multa
Versión 1.0 ( 29/01/2018 )
Autores ?
Fuentes ?
Dependencias Ninguno
Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando tengamos que pagar una multa.
Precondición PD
Secuencia normal Paso Acción
1 El sistema mira si han pasado 20 días desde la notificación.
2 Si no han pasado esos días el sistema le aplica el 50% de descuento a la cantidad de la multa., el sistema PD
3 El sistema pedirá los datos de la tarjeta bancaria:
- titular
- número
- fecha de caducidad
- código de seguridad
4 El actor Multado (ACT-0004) introduce sus datos bancarios
5 El sistema se comunica con el sistema bancario para hacer el pago.
Postcondición PD
Excepciones Paso Acción
5 Si no hay comunicación a los 30 segundos, el sistema cancela la operación, a continuación este caso de uso queda sin efecto
